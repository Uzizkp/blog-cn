Unicorn, 一个简洁且实用的Hexo主题

根据 butterfly hexo-theme-butterfly 主题魔改

示例 : 途中的树 | GitHub: hexo-theme-unicorn

有任何问题可以加入社区讨论 : 微信群 | Telegram

别忘了点个✨Star哦， 也欢迎提供赞助

本主题可以兼容Butterfly Docs大部分内容
相关设置文章：途中的树2.0食用指南

安装

  1. 如果你没有 pug 以及 stylus 的渲染器，请下载安装：
npm install hexo-renderer-pug hexo-renderer-stylus --save
  1. 在hexo根目录里执行
git clone -b master https://github.com/Uzizkp/hexo-theme-unicorn.git

其他特有设置，正在撰写教程，谨请期待 …

微博页面

主题配置了基于Twikoo的微博页面，示例
需要开启首先在主题Config中开启Twikoo评论，具体方法参照这里
新建Page，设置如下

---
title: 微博
date: 2021-06-12 21:04:45
top_img: false
type: 'weibo'
---
<div id="weibo-head">
    <a class="weibo-head-title">我的微博 .</a>
</div>
<script type="text/javascript">
	$("#weibo-head").on("click", function(e){
	    $(".tk-submit").show();
	    e.stopPropagation();
	});
</script>
